Weekly Workshops

Learn web development on campus at our Thursday night workshops.

We'll start off each workshop with a 30 minute lesson and you'll spend the next hour creating a personal website or a more complex web application.

Can't make it? Slides, notes, and videos will be posted online.

Week Topic Description Time Location
0 Welcome to HackBU Learn about HackBU and the topics we will cover. We'll expalin the difference between HTML, CSS, JavaScript, Ruby, and front-end vs. back-end development with an interactive demo. Jan. 30 at 8 p.m. LH 14
1 Introduction to HTML Learn how to create a very basic personal website in HTML. We will show the basic structure of every web page, various HTML tags, the difference between inline and block elements, and how to view the HTML source of a web page.
2 Introduction to CSS Begin styling your personal website as we explore the basics of CSS.
3 Advanced CSS Dive deeper into CSS and learn more advanced topics including floats, the box model, and grids.
4 Hack Night
5 JavaScript and jQuery Learn the basics of JavaScript and jQuery, the scripting language for Web pages.
6 Installfest A crash course in Ruby followed by a Ruby, Ruby on Rails, Git, and Heroku installfest. Extra volunteers will be on hand to help you install these programs!
7 MVC, Command Line, Git, and GitHub Learn about the popular Model-View-Controller pattern used by many web frameworks. We will also cover basics of the command line, Git, and GitHub.
8 Ruby on Rails Create the foundation of a web application using Ruby on Rails.
9 More Rails Continue building your web application with help from HackBU staff.
10 Hackathon
