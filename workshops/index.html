Weekly Workshops

Learn web development on campus at our Thursday night workshops.

We'll start off each workshop with a 30 minute lesson and you'll spend the next hour creating a personal website or a more complex web application.

Can't make it? Slides, notes, and videos will be posted online.

Portfolio Contest

We're launching a campus-wide contest and giving away two great prize packs to the students that create the best portfolios.

The contest will end the week after we finish covering HTML and CSS.

Insterested in sponsoring a prize? Please reach out to daniel@hackbu.org.

Schedule

Come to one of our weekly workshops and learn how to create websites.

Week Topic Description Time Location
0 Welcome to HackBU Learn about HackBU and the topics we will cover. We'll explain the difference between HTML, CSS, JavaScript, Ruby, and front-end vs. back-end development with an interactive demo. Jan. 30 at 8 p.m. LH 14
1 Introduction to HTML and CSS Learn how to create a very basic personal website in HTML. We will show the basic structure of every web page, various HTML tags, the difference between inline and block elements, and how to view the HTML source of a web page. We'll then move forward and learn how to style a web page using CSS. Feb. 20 at 8 p.m.

3 Advanced CSS Dive deeper into CSS and learn more advanced topics including floats, the box model, and grids. Mar. 6*

5 Installfest A crash course in Ruby followed by a Ruby, Ruby on Rails, Git, and Heroku installfest. Extra volunteers will be on hand to help you install these programs! Mar. 20*
6 MVC, Command Line, Git, and GitHub Learn about the popular Model-View-Controller pattern used by many web frameworks. We will also cover basics of the command line, Git, and GitHub. Mar. 27*
